Markdown Syntax

Note: The symbol ⎵ represents a standard space.

Line break markers
Headings

Level 3 = ###⎵ + Title
Level 4 = ####⎵ + Title
Level 5 = ##### ⎵ + Title
(Levels 1 and 2 are reserved for the blog title and post title)

Lists

Numbered = 1.⎵ + list item
Unnumbered = *⎵ + list item or -⎵ + list item

2nd level = ⎵⎵⎵⎵1.⎵ or ⎵⎵⎵⎵*⎵ / ⎵⎵⎵⎵-⎵ + list item
(indentation = 4 spaces)

Definitions/Descriptions

Term = term
Definition = :⎵ + definition/description

Miscellaneous

Horizontal line = ----

Preformatted text = ⎵⎵⎵⎵ + text

Quotation block = >⎵ + text

Abbreviations

*[ + abbreviation + ]: + title (optional)
Abbreviations are applied throughout the text and are then removed from the final result.

Block HTML

Insert the HTML code directly.

End-of-line markers

Line break = ⎵⎵ (2 spaces)

HTML attributes

Use { [.class] [#id] [attribute=value] } at the end of the (first) line
Valid for titles, images, links, code blocks.

Position-independent markers

Emphasis = * + text + *
Strong emphasis = ** + text + **

Link = ( + url + )
            [ + name + ]( + URL + )
            [ + name + ]( + url + “title” + )
[ + name + ]( + url + ){ lang= + language + }
            [ + name + ]( + url + “title” + ){ lang= + language + }

Image = ![ + name + ]( + url + )
No equivalent for more complex forms (positioned and/or with title, caption, etc.); in this case, use HTML

Code = ` + code here + `

Note = [^ + id + ]
[^ + id + ]:⎵ + Body of the note.

Literal = \ + character (prevents Markdown code from being interpreted)

Specific to Markdown
Code blocks

``` + [ language- + language ]
Code content
```

Tables

Example:

First Header  | Second Header
    ------------- | -------------
    Content Cell  | Content Cell
    Content Cell  | Content Cell
HTML and Markdown

An HTML block can contain content using Markdown syntax, provided that the markdown=“1” attribute is added to the block. In some cases, such as cell tables (where there is a risk of confusion), only inline elements will be interpreted. In this case, use markdown=“block” as the block attribute.
