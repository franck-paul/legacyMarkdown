Syntaxe Markdown

Note : Le symbole ⎵ représente un espace standard.

Marqueurs de retour à la ligne
Entêtes

Niveau 3 = ###⎵ + Titre
Niveau 4 = ####⎵ + Titre
Niveau 5 = ##### ⎵ + Titre
(Les niveaux 1 et 2 sont réservés au titre du blog et au titre de l'article)

Listes

Numéroté = 1.⎵ + élément de la liste
Non numéroté = *⎵ + élément de la liste ou -⎵ + élément de la liste

2e niveau = ⎵⎵⎵⎵1.⎵ ou ⎵⎵⎵⎵*⎵ / ⎵⎵⎵⎵-⎵ + élément de liste
(indentation = 4 espaces)

Définitions/Descriptions

Terme = terme
Définition = :⎵ + définition/description

Divers

Ligne horizontale = ----

Texte préformaté = ⎵⎵⎵⎵ + texte

Bloc de citation = >⎵ + texte

Abréviations

*[ + abréviation + ] : + titre (facultatif)
Les abréviations sont appliquées tout au long du texte et sont ensuite supprimées du résultat final.

Bloc HTML

Insérer directement le code HTML.

Marqueurs de fin de ligne

Retour à la ligne = ⎵⎵ (2 espaces)

Attributs HTML

Utilisez { [.class] [#id] [attribut=valeur] } à la fin de la (première) ligne.
Valable pour les titres, les images, les liens, les blocs de code.

Marqueurs indépendants de la position

Emphase = * + texte + *
Emphase forte = ** + texte + **

Lien = ( + url + )
[ + nom + ]( + URL + )
[ + nom + ]( + url + "titre" + )
[ + nom + ]( + url + ){ lang= + langue + }
[ + nom + ]( + url + "titre" + ){ lang = + langue + }

Image = ![ + nom + ]( + url + )
Pas d'équivalent pour les formes plus complexes (positionnées et/ou avec titre, légende, etc.) ; dans ce cas, utiliser HTML

Code = ` + code ici + `

Note = [^ + id + ]
[^ + id + ]:⎵ + Corps de la note.

Littéral = \ caractère (empêche l'interprétation du code Markdown)

Spécifique à Markdown
Blocs de code

``` + [ language- + language ]
Contenu du code
```

Tableaux

Exemple :

Premier en-tête | Deuxième en-tête
    ------------- | -------------
    Cellule de contenu | Cellule de contenu
    Cellule de contenu | Cellule de contenu
HTML et Markdown

Un bloc HTML peut contenir un contenu utilisant la syntaxe Markdown, à condition que l'attribut markdown="1" soit ajouté au bloc. Dans certains cas, comme les tableaux cellulaires (où il y a un risque de confusion), seuls les éléments en ligne seront interprétés. Dans ce cas, il convient d'utiliser markdown="block" comme attribut de bloc.
